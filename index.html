üöÄ Quick Start

Windows Users

  1. Download Hurricane-AutoInstall.bat
  2. Double-click the file to run
  3. The installer will automatically:
    • Check for Java
    • Download the launcher (~60KB)
    • Launch Hurricane
  4. On first launch, game files (~100MB) will be downloaded
  5. Enjoy Hurricane!

  1. Download Hurricane-AutoInstall.sh
  2. Make it executable and run:

  3. The installer will handle the rest automatically

‚ùì Troubleshooting

Launcher won't start

  • Verify Java is installed: java -version
  • Make sure both launcher.jar and launcher.hl are in the same folder
  • Try running with administrator privileges (Windows)

Game crashes or errors

  • Clear the launcher cache (usually in %LOCALAPPDATA%\Haven Launcher)
  • Ensure you have sufficient memory allocated
  • Check that your graphics drivers are up to date

Download fails

  • Check your internet connection
  • Verify GitHub Pages is not blocked by your firewall
  • Try again later (GitHub Pages may experience temporary issues)

üîÑ Updates

The launcher automatically checks for updates each time you start it. When new versions are available, they will be downloaded automatically.

To force a fresh download of all files, delete your launcher cache directory:

  • Windows: %LOCALAPPDATA%\Haven Launcher
  • Linux: ~/.cache/haven-launcher
  • macOS: ~/Library/Caches/Haven Launcher
